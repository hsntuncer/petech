Workday SAP Integration Package

INTERACTIVE APPLICATIONS

What is Workday Integration?

Workday - SAP integration is a difficult and troublesome integration in itself because it combines two different systems in terms of design, structure and change management.

Workday Integration Package Advantages

As Petech Consultancy, we have developed SAP - WORKDAY integration package on SAP by taking into account customer needs and needs with our experiences in previous projects

Integration Scheme


The following information can be transferred from Workday to SAP or from SAP to Workday;

  • Organization Management (OM)
  • Personnel Management (PA)
  • Payroll Information

Features of Workday Integration Package
  • Flexible integration structure
  • Integration process monitoring reports
  • Steps to map data between Workday and SAP
  • Production of IDOC in accordance with XML structure of workday
  • Logging of incoming data on IDOCs
  • PI / SFTP Integration
  • Identifying missing data in the workday and informing the related units by automatic mail
  • Informing the related units by e-mail in case of errors in each step of integration
  • Approval processes
  • Informational mails on the SAP side in recruitment and changes
